# PolyChat

PolyChat is a multi-model AI chat web application that allows users to interact with different large language models from a single interface. The project supports manual and automatic model selection along with persistent chat history.

This project was developed as part of a Coding Club Web Development task.

---

## Features

- Multi-model support (OpenAI, Google Gemini, Claude/Cohere)
- Manual model selection by the user
- Automatic model switching based on query type
- Persistent chat history using MongoDB
- Clean and minimal user interface
- Sidebar-based chat navigation

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB (Atlas)
- APIs: OpenAI, Google Gemini, Cohere / Claude
